## 📌 **Deskripsi**

**GhostTrack** adalah alat **diagnostik jaringan dan profiling sistem** generasi terbaru yang dirancang untuk para profesional IT, network engineer, dan security researcher. Dengan kemampuan **zero-footprint monitoring**, GhostTrack mampu mengumpulkan lebih dari **100+ parameter sistem** secara real-time tanpa meninggalkan jejak di perangkat target.

## 🔒 **Mengapa Script Ini Di-Encrypt?**

> ⚠️ **Catatan Penting:**
>
> Script ini menggunakan **3-layer encryption** (Base64 → XOR → Base64) untuk melindungi **API keys dan endpoint Telegram** dari deteksi oleh sistem keamanan seperti antivirus, firewall, atau IDS/IPS.
>
> **Alasan:**
> - 🛡️ **API Key Protection**: Mencegah API key Telegram terbaca oleh pihak yang tidak berwenang.
> - 🔐 **Security Through Obscurity**: Menghindari deteksi oleh signature-based antivirus.
> - 📡 **Endpoint Obfuscation**: Menyamarkan endpoint Telegram agar tidak terblokir oleh firewall.
> - 🚀 **Zero Detection**: Menghindari flagging oleh sistem keamanan enterprise.
